CNN Model Applied on the Dataset

The CNN(Convolutional Neural Network) is a class of deep learning neural networks. CNNs represent a huge breakthrough in image recognition. They’re most ordinarily wont to analyze visual imagery and are frequently working behind the scenes in image classification.


Basic Architecture

In a CNN architecture there are two main parts A convolution tool that separates and identifies the varied features of the image for analysis during a process called as Feature Extraction A fully connected layer that utilizes the output from the convolution process and predicts the category of the image supported the features extracted in previous stages.
